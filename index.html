Asistente InDrive

Asistente para InDrive: optimiza viajes bajo una configuración.

Descargar APK Ver en Google Play

Características

  • Bot para InDrive: Automatiza acciones repetitivas dentro de la app InDrive usando el servicio de accesibilidad.
  • Perfiles de zonas: Crea y administra perfiles que contienen tus zonas preferidas, zonas bloqueadas y configuraciones por perfil.
  • Zonas bloqueadas / Zonas Diamante / Zonas Gold: Define zonas en el mapa para bloquear solicitudes, o marcar zonas premium (Diamante/Gold) con reglas especiales.
  • Precio por km y tramos: Configura tarifas por kilómetro con múltiples tramos, precio mínimo, precio máximo y variantes inteligentes por distancia o aeropuerto.
  • Gestión de membresías: Sistema de membresía con días de suscripción, verificación y beneficios exclusivos.
  • Descargas y actualizaciones: UpdateManager y gestor de descargas para instalar actualizaciones (incluye FileProvider para instalación de APK).
  • Captura de pantalla y OCR: Funciones para capturar pantalla y procesar texto mediante OCR (cuando aplique).
  • Botón flotante: Servicio que muestra un botón flotante para controles rápidos mientras la app está activa.
  • Notificaciones push: Integración con Firebase Cloud Messaging para avisos y comunicaciones.
  • Multilenguaje: Soporte para varios idiomas (español, inglés, portugués, indonesio, árabe, tamazight, etc.).
  • Optimización y utilidades: Limpieza de caché, comprobación de permisos, opciones de configuración avanzada y referidos.

Novedades

  • Perfiles en zonas bloqueadas: Ahora puedes crear múltiples perfiles que guardan zonas bloqueadas y alternar entre ellos rápidamente.
  • Más tramos para precio por km: Se añadieron más tramos configurables para ajustar el precio por kilómetro según distancia/segmento.
  • Mejoras en Zonas Diamante/Gold: Nuevas opciones para definir radio y ajustes por zona premium.
  • Correcciones de bugs: Varias correcciones menores y estabilidad mejorada en el gestor de descargas y permisos.
  • Mejoras de rendimiento: Optimización en el consumo de memoria y tiempos de respuesta del servicio de accesibilidad.

Consulta los archivos README del proyecto para un registro de cambios completo.

Preguntas frecuentes

¿Es seguro descargar el APK?

Sí, descarga el APK desde este sitio únicamente. Comprueba que el nombre del paquete coincide con el que esperas (com.warfre.auto_click).

¿Necesito habilitar instalaciones desde orígenes desconocidos?

En Android 8+ suele pedirse permiso por app al instalar desde APK. Revisa la configuración de seguridad si es necesario.
